About project

Плагин позволяет посетителям сайта отправлять письма администрации посредством сайта.

Для администраторов:

  • Указание неограниченного количества e-mail адресов;
  • Анти-флуд контроль;
  • Настройка модуля посредством админ-панели;
  • Белый и черный список IP адресов.

System requirements

To use this plugin requires Livestreet CMS version 1.0 or later.

Simple Install

To install feedback do this:

  • Download the latest build
  • Unpack archive
  • Rename dir to feedback
  • Upload to FTP, in the plugins folder
  • Activate plugin
